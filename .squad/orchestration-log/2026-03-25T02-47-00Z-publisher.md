## Execution Summary

Publisher agent completed investigation into dashboard publish 500 error.

### Root Cause Identified
- Dashboard draft/publish routes require `substackService` dependency
- `startServer()` constructs `imageService` but does **not** construct/pass `SubstackService` to `createApp()`
- Result: Runtime wiring bug causes HTTP 500 even when `.env` contains valid `SUBSTACK_TOKEN` and `SUBSTACK_PUBLICATION_URL`

### UX Finding
