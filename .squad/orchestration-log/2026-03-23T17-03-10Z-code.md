## Technical Details

**Root Issue:** When `writeDraft()` encountered a malformed draft (e.g., missing TLDR), the immediate self-heal retry used only the upstream context (article facts, panel notes) without the failed draft body. This forced the model to essentially restart, often producing the same error or dropping working analysis.

**Fix:** Append the failed draft under a dedicated `## Failed Draft To Revise` section before the retry, allowing Writer to see and fix what it just produced.

**Consistency:** Editor and Publisher now explicitly treat TLDR/structure misses as `REVISE` cases in their guidance, framing revisions as precision improvements rather than wholesale rewrites.
