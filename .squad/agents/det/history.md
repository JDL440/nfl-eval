### Strategic Assessment
_Last updated: 2026-03-12 by DET agent_

**Window Status:** 📊 Still open but narrowing. Core of Goff/ARSB/Sewell/Hutchinson/McNeill is elite. 9-8 was an injury-driven regression, not a talent problem.

**Biggest Risks:**
1. 🚨 No viable EDGE opposite Hutchinson — if he's doubled, pass rush dies
2. 🚨 LT vacancy is alarming — Goff's blindside unprotected
3. ❓ Branch + Joseph health — if either can't go Week 1, secondary is in trouble
4. 📊 Ben Johnson in Chicago knows every Lions tendency — division games will be chess matches

**Key Decisions Ahead:**
1. R1 #17: EDGE or OT? This pick defines the offseason.
2. Hutchinson extension timing — lock him up before the EDGE market explodes further ($40M+ AAV)
3. Joseph extension — All-Pro safety, needs to be signed before 5th-year option expires
4. Secondary FA signing — need one more veteran S or CB before camp

**Cap Flexibility:** With ~$36M in space and $6.5M in draft pool allocated, Lions have ~$29M for remaining moves. Enough for 1 mid-tier starter + 3-4 depth signings.
