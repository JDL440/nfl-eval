- Run game: Hybrid zone + gap blocking scheme
- Inside zone, outside zone, counter, power — all in the playbook
- Heavy use of tight ends, play-action, RPOs
- Physical, ground-first identity behind elite offensive line
- Under-center sets mixed with shotgun — pro-style versatility

**Seattle System (OC Brian Fleury):**
- Shanahan coaching tree (came from SF 49ers as run game coordinator)
- Base: Outside zone + inside zone with gap concept integration
- Heavy use of motion, misdirection, tight ends/fullbacks
- Committee back approach — multiple backs, varied personnel groupings
- Backs expected to contribute in zone AND gap schemes, plus passing game
- Physical, fast tempo, aggressive red zone run philosophy

**Scheme Translation:** 🟢 STRONG FIT
- Notre Dame's zone + gap hybrid is structurally similar to Fleury's Shanahan-derived system
- Price already has experience in both zone (read the block, make one cut) and gap (follow the puller, hit the hole) concepts
- His burst and vision in zone runs translate directly to Seattle's outside zone attack
- Committee back experience at ND maps perfectly to Seattle's multi-back approach
- Receiving work is limited (15 career catches) but growing — area for NFL development
- Pass protection willingness noted by scouts — critical for Shanahan-tree systems
- Kick return value adds a dimension Seattle could exploit on special teams

**Key Translation Concerns:**
- ⚠️ Pass protection technique needs refinement (limited reps at ND due to committee role)
- ⚠️ Route tree is underdeveloped — only 15 career catches. Shanahan backs need to catch
- ⚠️ Ball security — 3 fumbles in 2025 flagged by coaching staff as development area
- ⚠️ Never carried 20+ times in a game — durability as a heavier-workload back is unproven
