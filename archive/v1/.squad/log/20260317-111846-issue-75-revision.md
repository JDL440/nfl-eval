## Root Causes

1. **Clipping (tests 3/4/5):** `estimateRowHeight` used desktop-calibrated char-width constants (17px). Mobile 22px font produces wider characters → more wrapping → taller rows. Canvas height underestimated.
2. **Header collision (test 6):** `table-layout: fixed` forces column widths, but `thead th` lacked `overflow-wrap`. Header text overflowed column boundaries. Compounded by `text-transform: uppercase` + `letter-spacing: 0.08em`.
